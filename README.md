#GenAI – Newspaper Article Generator

This project is a context-aware Newspaper Article Generator built using Retrieval-Augmented Generation (RAG) and LangChain, integrated with OpenAI’s GPT models to produce factual, coherent, and human-like news articles.

#Key Features

Context-Aware Article Generation
Generates high-quality newspaper-style articles using GPT models with RAG for improved factual grounding.
Retrieval-Augmented Generation (RAG)
Fetches contextual information from Wikipedia and uploaded documents, ensuring accuracy and relevance.
LangChain Workflow Design
Built LLM pipelines for document retrieval, prompt structuring, and article generation.
Agentic AI Exploration
Experimented with multi-agent collaboration frameworks to enhance automation and reasoning.
